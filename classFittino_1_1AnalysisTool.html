Detailed Description

Abstract class which provides basic functionality for analysis tools like optimizers or samplers.

This class contains auxiliary objects needed by every analysis tool such as the tool's name,
a pointer to the model to be analysed, a counter for the number of iteration steps and a
random number generator.
