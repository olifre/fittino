Detailed Description

Fittino exception base class.

Exceptions make up the backbone of the error handling system in Fittino. This class provides
basic functionality for the various Fittino exception classes. Their main purpose is to
print out messages with further information about the occurred exception in a uniform way.
Supported Fittino exceptions are

  • CalculatorException
  • ConfigurationException
  • ExecutorException
  • InputException
  • OptimizerException
  • RedirectorException
  • SLHAFileException
  • TimeoutExecutorException
