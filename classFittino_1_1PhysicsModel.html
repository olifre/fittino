◆ Initialize()

void Fittino::PhysicsModel::Initialize ( )
protectedvirtual

This function is mainly used to provide a dedicated command to print the model
configuration to the screen and is usually called at the end of the model constructor
after the actual "initialization" of the model is already finished. Therefore, it should
not be used to alter the state of the model. However, since third party code (like e.g.
model calculators) sometimes does not differ between initialization and printing, this
is also the place where third party code is initialized.
